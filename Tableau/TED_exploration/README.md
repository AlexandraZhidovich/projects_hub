# Историческое исследование TED-конференций

## Описание проекта
*TED* (аббревиатура от англ. technology, entertainment, design — технологии, развлечения, дизайн) — 
американский частный некоммерческий фонд, известный прежде всего своими ежегодными конференциями, нацеленными на распространение идей. 
Выступления в формате TED — это яркие 17‑минутные речи, которые побуждают аудиторию к размышлениям. 
Они затрагивают как масштабные мировые вызовы, так и повседневные проблемы, актуальные для многих.

*Данные*, предоставленные для исследования, были собранны на основе материалов сайта TED [https://www.ted.com/].

## Описание данных
Данные выступлений хранятся в трех csv-файлах с одинаковой структурой:
- talk_id — идентификатор выступления;
- url — ссылка на запись выступления;
- title — название выступления;
- description — краткое описание;
- film_date — дата записи выступления;
- duration — длительность в секундах;
- views — количество просмотров;
- main_tag — основная категория, к которой относится выступление;
- speaker_id — уникальный идентификатор автора выступления;
- laughter_count — количество раз, когда аудитория смеялась в ходе выступления;
- applause_count — количество раз, когда аудитория аплодировала в ходе выступления;
- language — язык, на котором велось выступление;
- event_id — уникальный идентификатор конференции.

Справочник конференций в формате .csv содержит:
- conf_id — уникальный идентификатор конференции;
- event — название конференции;
- country — страна проведения конференции.

Справочник авторов выступления в формате .csv содержит:
- author_id — уникальный идентификатор автора выступления;
- speaker_name — имя автора;
- speaker_occupation — профессиональная область автора;
- speaker_description — описание профессиональной деятельности автора.
